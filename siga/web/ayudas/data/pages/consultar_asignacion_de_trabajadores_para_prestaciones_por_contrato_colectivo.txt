====== Operación Consultar ======  

Para **Consultar** el registro de **Asignación de Trabajadores por Contrato Colectivo** debe seguir los siguientes pasos:   <html><br><br></html>

  * Presione el **Botón**  {{:btnlista.jpg|}}    <html><br><br></html>

  * Seleccione de la **Lista** el **Código del Tipo de Contrato Colectivo** para el cual desea consultar el Registro de Asignación de Trabajadores para Prestaciones.

<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperasidetraparapreporconcolmod1.jpg|}}"></div><br><br></html>

  * El sistema automáticamente muestra el Registro de Asignación de Trabajadores para Prestaciones.<html><br><br></html>

  * Si desea consultar el **Registro de Asignación de Trabajadores para Prestaciones** de un nuevo **Contrato Colectivo** debe presionar nuevamente el** Botón** {{:btnlista.jpg|}}  y repetir los pasos anteriores. <html><br><br></html>



**Para consultar datos desde el Inicio de la pantalla de Asignación de Trabajadores para Prestaciones por Contrato Colectivo:**  <html><br><br></html>

  * El sistema muestra la **Lista** de todos aquellos **Tipos de Contratos Colectivos** que poseen asignados trabajadores para Prestaciones.<html><br><br></html>

  * Seleccione de la **Lista** el **Código del Tipo de Contrato Colectivo** para el cual desea consultar la **Asignación de Trabajadores para Prestaciones**. 

<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperasidetraparapreporconcolmod1.jpg|}}"></div><br><br></html>

  * El sistema automáticamente muestra la **Asignación de Trabajadores para Prestaciones** del **Contrato Colectivo** seleccionado.<html><br><br></html>
