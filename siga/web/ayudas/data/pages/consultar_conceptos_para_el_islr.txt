====== Operación Consultar ======

**Para consultar los Conceptos para el ISLR asociados a una determinada Nómina utilizando el Botón Lista:**   <html><br><br></html>

  * Presione el **Botón**  {{:btnlista.jpg|}}    <html><br><br></html>

  * Seleccione de la **Lista** el **Tipo de Nómina** para la cual desea consultar los Conceptos para el ISLR.
<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperconimpislrmod.jpg|}}"></div><br></html>

  * El sistema automáticamente muestra los **Conceptos para el ISLR** del  **Tipo de Nómina** seleccionado.   <html><br><br></html>

  * Si desea consultar los Conceptos de un nuevo Tipo de Nómina debe presionar nuevamente el** Botón** {{:btnlista.jpg|}}  y repetir los pasos anteriores.
 
<html><br><br></html>


**Para consultar datos desde el Inicio de la pantalla de Conceptos para el ISLR:**  <html><br><br></html>

  * El sistema muestra la **Lista** de **Conceptos para el ISLR** por cada nómina.
<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperconimpislrmod.jpg|}}"></div><br></html>

  * Seleccione de la Lista el **Tipo de Nómina** para la cual desea consultar los Conceptos para el ISLR..  <html><br><br></html>

  * El sistema automáticamente muestra los Conceptos para el ISLR asignados al Tipo de Nómina seleccionado.
