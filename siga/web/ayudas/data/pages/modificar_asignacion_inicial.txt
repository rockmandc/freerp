====== Modificar Registro ======

Para Modificar una **Asignación Inicial** ya definida: <html><br><br></html>

  * Presione el Botón de Consulta {{:btncatalogo.jpg|}} <html><br><br></html>

  * El sistema muestra un Catálogo con las Asignaciones Iniciales registradas.

<html><br><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:catalogo1.jpg|}}"></div><br><br></html>

  * El sistema muestra los **Datos del Código Presupuestario** y los **Datos de la Asignación** <html><br><br></html>

  * El único dato que podrá modificar una vez definida la Asignación Inicial de un determinado Código Presupuestario es la **Descripción** del **Código Presupuestario**.
