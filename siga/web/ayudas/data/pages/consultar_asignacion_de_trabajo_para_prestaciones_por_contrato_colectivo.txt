====== Operación Consultar ======

**Para consultar las Asignaciones de Trabajadores para Prestaciones por Contrato Colectivo a través del Botón Lista:**   <html><br><br></html>

  * Presione el **Botón**  {{:btnlista.jpg|}}    <html><br><br></html>

  * Seleccione de la **Lista** el **Tipo de Contrato Colectivo** para el cual desea consultar la Asignación de Trabajadores para Prestaciones por Contrato Colectivo. <html><br><br></html> 

<html><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperasidetraparapreporconcolmod1.jpg|}}"></div><br></html>

  * El sistema automáticamente muestra las Asignaciones de Trabajadores para Prestaciones por Contrato Colectivo.<html><br><br></html>

  * Si desea consultar la Asignación de Trabajadores para Prestaciones de un nuevo Contrato Colectivo debe presionar nuevamente el** Botón** {{:btnlista.jpg|}}  y repetir los pasos anteriores. <html><br><br></html>


**Para consultar datos desde el Inicio de la pantalla de Asignación de Trabajadores para Prestaciones por Contrato Colectivo:**  <html><br><br></html>

  * El sistema muestra la **Lista** con las Asignaciones de Trabajadores para Prestaciones por Contrato Colectivo.<html><br><br></html>

<html><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperasidetraparapreporconcolmod1.jpg|}}"></div><br></html>

  * Seleccione de la **Lista** el **Tipo de Contrato Colectivo** para el cual desea consultar la Asignación de Trabajadores para Prestaciones por Contrato Colectivo. <html><br><br></html>

  * El sistema automáticamente muestra las Asignaciones de Trabajadores para Prestaciones por Contrato Colectivo.<html><br><br></html>
