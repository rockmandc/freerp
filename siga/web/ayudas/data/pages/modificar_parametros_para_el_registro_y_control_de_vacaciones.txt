====== Operación Modificar ======

Para la operación de **Modificar** debe:  <html><br><br></html>

  * Seleccionar de la **Lista** el **Tipo de Nómina** para la cual desea modificar los parámetros para el registro y control de vacaciones. <html><br><br></html>

  * El sistema muestra los datos del Tipo de Nómina seleccionada. <html><br><br></html>

  * Las modificaciones que podrá realizar es quitar parámetros que ya esten asociados al tipo de nómina seleccionado a través del Botón {{:btnquitar.jpg|}}.<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/"></div><br></html>

  * Presionar {{:btnguardar.jpg|}} para actualizar la información.  <html><br><br></html> 

  * El sistema emite un **Mensaje de Éxito**
