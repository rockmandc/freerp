====== Operación Incluir ======

Para la operación de Registrar:

  * Presione el Botón  {{:btnnuevo.jpg|}}   <html><br><br></html>

  * Seguidamente ingrese el **Código del Banco** que se está definiendo. <html><br><br></html>

  * Ingrese el **Nombre** que haga referencia al Banco.  
<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:nomperdefbaninc.jpg|}}"></div><br></html>

  * Presionar {{:btnguardar.jpg|}} para registrar lo datos. <html><br><br></html>

  * El sistema emite un **Mensaje de Éxito**.
<html><br><div align="center"><img src="/ayudas/lib/exe/fetch.php/{{:msjexito.jpg|}}"></div><br></html>

  * Para verificar que el registro se guardó con éxito podemos consultarlo directamente desde la Lista.
