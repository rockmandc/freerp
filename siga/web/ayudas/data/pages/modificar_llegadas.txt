====== Operación Modificar ======

Para **Modificar** una determinada **Llegada**: <html><br><br></html>

  * Seleccionar de la ** Lista** el **Código del Empleado** para el cual desea modificar los datos de la llegada. <html><br><br></html>

  * Realice las modificaciones necesarias. Es importante destacar que solo podrá modificar el **Número de Días** y la **Observación**. <html><br><br></html>

  * Una vez realizadas las modificaciones debe presionar el Botón  {{:btnguardar.jpg|}} para actualizar la información. <html><br><br></html>
 
  * El sistema emite un **Mensaje de Èxito**.
